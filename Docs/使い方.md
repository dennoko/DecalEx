# DecalEx 使い方ガイド

lilToon拡張シェーダー「DecalEx」の使い方を説明します。

## 概要

DecalExは、lilToonにデカール専用スロットを追加する拡張シェーダーです。
UV上の任意の位置・サイズ・角度でデカールを貼り付けられ、スロットごとに以下の機能を組み合わせることができます。

- **メインデカールテクスチャ** — ブレンドモード付きカラーデカール
- **マスク** — デカール範囲を画像で細かく制御
- **ノーマルマップ** — デカール範囲に適用するノーマルマップ
- **MatCap** — デカール範囲に適用するMatCap
- **エミッション** — アニメーション対応の自己発光

スロットは **Decal Slot 1 / Slot 2** の2枠あり、それぞれ独立して設定できます。

## インストール

1. lilToonがプロジェクトにインストールされていることを確認
2. unitypackageをインポート
3. Unityがスクリプトを再コンパイルするのを待つ

## シェーダーの適用

1. マテリアルを選択
2. Shaderドロップダウンから `dennoko_decalex > lilToon` を選択
3. インスペクター下部の「Decal Slot 1」「Decal Slot 2」を展開して設定

描画モードは不透明・カットアウト・半透明に対応しています。

---

## スロット共通の構造

各スロット（Slot 1 / Slot 2）は同じ構造を持ちます。

### 基本

| パラメータ | 説明 |
|---|---|
| **Enable** | スロット全体のON/OFF |
| **Disable on Backface** | メッシュ裏面でデカールを無効化（全サブ機能に適用） |

---

### Position — 全サブ機能で共有

位置・スケール・角度は全サブ機能（テクスチャ・マスク・ノーマルマップ・MatCap・エミッション）で共通です。

| パラメータ | 説明 |
|---|---|
| **Offset X / Y** | UV空間でのデカール中心座標（0.5, 0.5 = UV中央） |
| **Scale X / Y** | デカールのUV幅・高さ（1.0 = UV全幅） |
| **Angle** | デカールの回転角度（-180〜180度） |

**設定例：** UV (0.3, 0.7) に幅0.2・高さ0.15のデカールがある場合
→ Offset X = 0.3、Offset Y = 0.7、Scale X = 0.2、Scale Y = 0.15

---

### Main Decal Texture

| パラメータ | 説明 |
|---|---|
| **Texture** | デカールのメインテクスチャ（Aチャンネルが透明度） |
| **Color** | テクスチャに乗算するカラー |
| **Opacity** | デカールの不透明度（テクスチャのAチャンネルとも乗算） |
| **Blend Mode** | ベースカラーとの合成方法（下表参照） |

#### ブレンドモード

| モード | 効果 |
|---|---|
| Add | 加算（明るくなる） |
| Screen | スクリーン（自然な明るさ） |
| Multiply | 乗算（暗くなる） |
| Overlay | オーバーレイ（コントラスト強調） |
| Soft Light | ソフトライト（穏やかな明暗） |
| Replace | 置換（デカール色で上書き） |
| Subtract | 減算（暗くなる） |
| Lighten | 比較(明)（明るい方を採用） |
| Darken | 比較(暗)（暗い方を採用） |

---

### Mask

| パラメータ | 説明 |
|---|---|
| **Use Mask** | マスクテクスチャを使用して適用範囲を限定 |
| **Mask Texture** | 範囲マスク（白=適用・黒=非適用）。位置はPositionと共有 |

マスクの外側では全サブ機能（テクスチャ・ノーマルマップ・MatCap・エミッション）が無効になります。

---

### Normal Map

| パラメータ | 説明 |
|---|---|
| **Enable Normal Map** | デカール範囲内にのみ適用するノーマルマップを有効化 |
| **Normal Map** | タンジェント空間ノーマルマップテクスチャ |
| **Strength** | ノーマルマップの強度（0=フラット、1=通常、3=強調） |

---

### MatCap

| パラメータ | 説明 |
|---|---|
| **Enable MatCap** | MatCapを有効化 |
| **Texture** | MatCapテクスチャ（球体の画像） |
| **Color** | MatCapに乗算するカラー |
| **Opacity** | MatCapの不透明度 |
| **Blend Mode** | ベースカラーとの合成方法（Main Decal Textureと同じ選択肢） |
| **Main Color Strength** | メインテクスチャ色をMatCapに乗算する強度（0=無効、1=完全） |
| **Blur** | MatCapのぼかし強度（MipMap必要） |
| **Normal Strength** | 法線マップの影響度（MatCap UV計算に使用） |
| **Use Reflection** | 視線反射モード（ON=鏡面反射的な挙動） |
| **Enable Lighting** | ライティング影響度（0=無効、1=完全適用） |
| **Shadow Strength** | 影部分でのMatCap減衰（0=影響なし、1=影で完全消滅） |
| **Rim Power** | リムマスク強度（正=エッジ、負=中心、0=無効） |
| **Emission Addition** | MatCapの色をEmissionとして加算する強度 |

#### Use Reflection について

- **OFF（通常）**: カメラ方向によらず法線に基づいた固定MatCap
- **ON（反射）**: カメラ方向によってMatCapの見え方が変化（鏡面反射的）

#### Rim Power について

| 値 | 効果 |
|---|---|
| 0 | リムマスク無効（全体に均等適用） |
| 正の値 | エッジ（縁）に強く適用。値が大きいほど縁のみに集中 |
| 負の値 | 中心に強く適用。絶対値が大きいほど中心のみに集中 |

---

### Emission

| パラメータ | 説明 |
|---|---|
| **Enable Emission** | エミッションを有効化 |
| **Texture** | エミッションテクスチャ（未割当の場合はフラットカラーとして発光） |
| **Color** | テクスチャに乗算するカラー |
| **Strength** | エミッション強度 |
| **Opacity** | 発光色をライティング結果へ重ねる強さ（0=暗い時に元の見た目を維持） |

#### Sin Wave（正弦波アニメーション）

エミッション強度をなめらかに振動させます。

| パラメータ | 説明 |
|---|---|
| **Sin Wave** | ON/OFF |
| **Speed** | 振動速度（大きいほど速い） |
| **Min** | 最小輝度（0=完全消灯） |
| **Max** | 最大輝度（1=Strength全開） |

#### Random Pulse（ランダムパルス）

強度をランダムにパルス発光させます。2系統のパターンをORで合成し、より複雑な点滅を表現できます。

| パラメータ | 説明 |
|---|---|
| **Random Pulse** | ON/OFF |
| **Pulse Probability 1** | パターン1が各時間ウィンドウで発光する確率（0〜1） |
| **Pulse Duration 1** | パターン1の発光持続時間（秒） |
| **Pulse Probability 2** | パターン2が各時間ウィンドウで発光する確率（0〜1） |
| **Pulse Duration 2** | パターン2の発光持続時間（秒） |

どちらかのパターンが発光条件を満たしたとき発光します（OR合成）。

#### UV Scroll（スクロールアニメーション）

エミッションテクスチャおよびスクロールマスクのUVを時間とともにスクロールさせます。

| パラメータ | 説明 |
|---|---|
| **UV Scroll** | ON/OFF |
| **Scroll X** | 横スクロール速度（UV/秒、-3〜3） |
| **Scroll Y** | 縦スクロール速度（UV/秒、-3〜3） |
| **Scroll Mask** | スクロールとともに動くマスクテクスチャ（白=発光・黒=非発光）。Tilingを設定するとパターンが繰り返される |

エミッションテクスチャが未割当の場合、スクロールマスクのみでスクロールアニメーションを表現できます。
エミッションテクスチャを設定した場合は、テクスチャとスクロールマスクが同じ速度でスクロールします。

Sin Wave・Random Pulse・UV Scrollは同時に有効化して組み合わせることができます。

## 描画モード

マテリアル上部の「Rendering Mode」から選択できます。

| モード | 説明 |
|---|---|
| **Opaque** | 不透明 |
| **Cutout** | アルファテスト（切り抜き） |
| **Transparent** | 半透明 |
