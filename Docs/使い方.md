# DecalEx 使い方ガイド

lilToon拡張シェーダー「DecalEx」の使い方を説明します。

## 概要

DecalExは、lilToonにデカール専用スロットを追加する拡張シェーダーです。
UV上の任意の位置・サイズ・角度でデカールを貼り付けられ、スロットごとに以下の機能を組み合わせることができます。

- **メインデカールテクスチャ** — ブレンドモード付きカラーデカール
- **マスク** — デカール範囲を画像で細かく制御
- **ノーマルマップ** — デカール範囲に適用するノーマルマップ
- **MatCap** — デカール範囲に適用するMatCap
- **エミッション** — アニメーション対応の自己発光

スロットは **デカールスロット 1 / スロット 2** の2枠あり、それぞれ独立して設定できます。

## インストール

1. lilToonがプロジェクトにインストールされていることを確認
2. unitypackageをインポート
3. Unityがスクリプトを再コンパイルするのを待つ

## シェーダーの適用

1. マテリアルを選択
2. Shaderドロップダウンから `dennoko_decalex > lilToon` を選択
3. インスペクター下部の「Decal Slot 1」「Decal Slot 2」を展開して設定

描画モードは不透明・カットアウト・半透明に対応しています。

---

## スロット共通の構造

各スロット（スロット 1 / スロット 2）は同じ構造を持ちます。

### 基本

| パラメータ | 説明 |
|---|---|
| **有効** | スロット全体のON/OFF |
| **裏面で無効** | メッシュ裏面でデカールを無効化（全サブ機能に適用） |

---

### 位置設定 — 全サブ機能で共有

位置・スケール・角度は全サブ機能（テクスチャ・マスク・ノーマルマップ・MatCap・エミッション）で共通です。

| パラメータ | 説明 |
|---|---|
| **オフセット X / Y** | UV空間でのデカール中心座標（0.5, 0.5 = UV中央） |
| **スケール X / Y** | デカールのUV幅・高さ（1.0 = UV全幅） |
| **角度** | デカールの回転角度（-180〜180度） |

**設定例：** UV (0.3, 0.7) に幅0.2・高さ0.15のデカールがある場合
→ オフセット X = 0.3、オフセット Y = 0.7、スケール X = 0.2、スケール Y = 0.15

---

### メインデカールテクスチャ

| パラメータ | 説明 |
|---|---|
| **テクスチャ** | デカールのメインテクスチャ（Aチャンネルが透明度） |
| **カラー** | テクスチャに乗算するカラー |
| **不透明度** | デカールの不透明度（テクスチャのAチャンネルとも乗算） |
| **ブレンドモード** | ベースカラーとの合成方法（下表参照） |

#### ブレンドモード

| モード名 | 効果 |
|---|---|
| 加算 | 加算（明るくなる） |
| スクリーン | スクリーン（自然な明るさ） |
| 乗算 | 乗算（暗くなる） |
| ソフトライト | ソフトライト（穏やかな明暗） |
| 置換 | 置換（デカール色で上書き） |

---

### マスク

| パラメータ | 説明 |
|---|---|
| **マスク使用** | マスクテクスチャを使用して適用範囲を限定 |
| **マスクテクスチャ** | 範囲マスク（白=適用・黒=非適用）。位置は位置設定と共有 |

マスクの外側では全サブ機能（テクスチャ・ノーマルマップ・MatCap・エミッション）が無効になります。

---

### ノーマルマップ

| パラメータ | 説明 |
|---|---|
| **ノーマルマップ有効** | デカール範囲内にのみ適用するノーマルマップを有効化 |
| **ノーマルマップ** | タンジェント空間ノーマルマップテクスチャ |
| **強度** | ノーマルマップの強度（0=フラット、1=通常、3=強調） |

---

### MatCap

| パラメータ | 説明 |
|---|---|
| **MatCap有効** | MatCapを有効化 |
| **テクスチャ** | MatCapテクスチャ（球体の画像） |
| **カラー** | MatCapに乗算するカラー |
| **不透明度** | MatCapの不透明度 |
| **ブレンドモード** | ベースカラーとの合成方法（メインデカールテクスチャと同じ選択肢） |
| **メインカラー強度** | メインテクスチャ色をMatCapに乗算する強度（0=無効、1=完全） |
| **ぼかし** | MatCapのぼかし強度（MipMap必要） |
| **法線強度** | 法線マップの影響度（MatCap UV計算に使用） |
| **反射モード** | 視線反射モード（ON=鏡面反射的な挙動） |
| **ライティング有効** | ライティング影響度（0=無効、1=完全適用） |
| **シャドウ強度** | 影部分でのMatCap減衰（0=影響なし、1=影で完全消滅） |

#### 反射モード について

- **OFF（通常）**: カメラ方向によらず法線に基づいた固定MatCap
- **ON（反射）**: カメラ方向によってMatCapの見え方が変化（鏡面反射的）

---

### エミッション

| パラメータ | 説明 |
|---|---|
| **エミッション有効** | エミッションを有効化 |
| **テクスチャ** | エミッションテクスチャ（未割当の場合はフラットカラーとして発光） |
| **カラー** | テクスチャに乗算するカラー |
| **強度** | エミッション強度 |
| **不透明度** | 発光色をライティング結果へ重ねる強さ（0=暗い時に元の見た目を維持） |

#### サイン波（正弦波アニメーション）

エミッション強度をなめらかに振動させます。

| パラメータ | 説明 |
|---|---|
| **サイン波** | ON/OFF |
| **速度** | 振動速度（大きいほど速い） |
| **最小** | 最小輝度（0=完全消灯） |
| **最大** | 最大輝度（1=強度全開） |

#### ランダムパルス

強度をランダムにパルス発光させます。2系統のパターンをORで合成し、より複雑な点滅を表現できます。

| パラメータ | 説明 |
|---|---|
| **ランダムパルス** | ON/OFF |
| **発光確率1** | パターン1が各時間ウィンドウで発光する確率（0〜1） |
| **発光時間1** | パターン1の発光持続時間（秒） |
| **発光確率2** | パターン2が各時間ウィンドウで発光する確率（0〜1） |
| **発光時間2** | パターン2の発光持続時間（秒） |

どちらかのパターンが発光条件を満たしたとき発光します（OR合成）。

#### UVスクロール（スクロールアニメーション）

エミッションテクスチャおよびスクロールマスクのUVを時間とともにスクロールさせます。

| パラメータ | 説明 |
|---|---|
| **UVスクロール** | ON/OFF |
| **スクロール X** | 横スクロール速度（UV/秒、-3〜3） |
| **スクロール Y** | 縦スクロール速度（UV/秒、-3〜3） |
| **スクロールマスク** | スクロールとともに動くマスクテクスチャ（白=発光・黒=非発光）。Tilingを設定するとパターンが繰り返される |

エミッションテクスチャが未割当の場合、スクロールマスクのみでスクロールアニメーションを表現できます。
エミッションテクスチャを設定した場合は、テクスチャとスクロールマスクが同じ速度でスクロールします。

サイン波・ランダムパルス・UVスクロールは同時に有効化して組み合わせることができます。

---

## 描画モード

マテリアル上部の「Rendering Mode」から選択できます。

| モード | 説明 |
|---|---|
| **Opaque** | 不透明 |
| **Cutout** | アルファテスト（切り抜き） |
| **Transparent** | 半透明 |

> **注意:** Cutout / Transparent モードでは、シェーダー命令数の制限により **デカールスロット 2 のエミッション** は使用できません。スロット 2 のエミッションが必要な場合は Opaque モードをご使用ください。

---

## 重要な注意点

- アバターアップロード後など、Unityのシーン上でこのシェーダーを適用しているマテリアルが透明になることがあります。対象マテリアルを選択し、Inspectorで「Decal Slot 2」を展開して下部にある「lilToon Refresh Shader」ボタンを押してください。ボタンが表示されない場合や選択ができない場合は、Unity上部メニューの Assets > lilToon > Refresh Shader を実行してください。
- アルファオーバーライドやデカールのカラーで黒を指定しても、後続のシェーダー処理によって必ずしもそのままの色で出力されるとは限りません。特に暗い色は他の合成やライティングで見えにくくなることがあるため、十分に確認してください。


## アルファオーバーライド（Cutout / Transparent 専用）

Cutout / Transparent モードでのみ使用できる機能です。デカールのマスク範囲内のアルファ値を任意の固定値に上書きできます。

| パラメータ | 説明 |
|---|---|
| **アルファオーバーライド有効** | ON/OFF |
| **アルファ値** | デカール範囲に適用する固定アルファ値（0=透明、1=不透明） |

マスクの濃淡に応じて元のアルファ値と設定値の間を補間します。Opaque モードではこのパラメータは表示されません。
